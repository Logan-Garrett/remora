Based on the context shared in this session, help diagnose the bug.

1. **Reproduce** — what are the exact steps to trigger the issue?
2. **Symptoms** — what happens vs. what should happen?
3. **Hypotheses** — list 2-3 possible root causes, ranked by likelihood
4. **Next steps** — for each hypothesis, what would confirm or rule it out?
5. **Quick fix** — if the cause is obvious, suggest a fix with the exact code change

Be specific. Reference file paths and line numbers from the shared context.
